Projects

My personal and collaborative projects outside of school, showcasing my skills in graphic design, animation, and coding.

Pixel Pirate

Pixel Pirate is a 2D video game created by Maxim Nyanin, Adam Kanoun, and Shrivardhan Mishra for UTM PyJaC hackathon 2022. This is a player versus computer game in which the player has to defend against an ever-increasing barrage of cannonballs shot at their ship. The player uses keys to maneuver the pirate character, who uses a shield to block the incoming projectiles. Endurance is key here! We used Python in a modular fashion to create the various aspects of the game, from graphics to cannonball motion.

To-Do List

A simple to-do list web application that allows to add activities to the list and remove them once completed. Written using HTML, SASS/CSS, and Vanilla JavaScript. This to-do list web app is built to be responsive with mobile first in mind. It also includes accessibility (A11y) features for keyboard navigation and screen readers. Finally, it utilizes the Web Storage API to create a persistent data store so the list items reappear any time user visit the web app.

Space Shooter

Space Shooter is a 2D game written in Python using pygame library. The player gets to control a spaceship using WASD controls, shoot lazer projectiles, and avoid being hit by the enemie fire. This game is endless with the goal of surviving enemy waves and preventing them from crossing the bottom of the screen. As the game goes on, the level counter gets updated showing the current wave of enemies as more of them gets spawned, thus making the game more challenging.
